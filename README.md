# Produktivitetsverktyg för smart arbetsdag

## Projektbeskrivning
Detta projekt är ett produktivitetsverktyg som hjälper användare att optimera sin arbetsdag genom tidsspårning, energiprediktion och smarta rekommendationer för arbete och pauser.

Applikationen är utformad för att stödja ett hållbart och effektivt arbetssätt genom att anpassa arbetsblock och pauser efter användarens energinivåer under dagen.

## Syfte och mål
Målet med applikationen är att hjälpa användare att:
- Tidsspåra sitt arbete med hög precision
- Logga och visualisera energinivåer över dagen
- Få rekommendationer för optimala arbetsblock
- Ta pauser vid rätt tillfällen
- Växla mellan olika fokuslägen för olika typer av arbete

## Funktioner
- **Tidsspårning** av arbetsuppgifter och aktiviteter  
- **Energiloggar** med visuell översikt över dagens energinivåer  
- **Smarta rekommendationer** för när användaren bör arbeta eller vila  
- **Fokuslägen**:
  - Deep Work
  - Möte
  - Paus

## Plattformar
Projektet levereras som:
- **Webbapplikation** byggd i React
- **Mobilapplikation** byggd i React Native

Affärslogik och kärnfunktionalitet delas mellan plattformarna för att säkerställa konsekvent beteende och effektiv utveckling.

## Teknikstack
- React (Webb)
- React Native (Mobil)
- Delad logik mellan plattformar
